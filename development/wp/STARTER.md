# WordPress starter

This directory contains WordPress 7.0 and a minimal block theme named `starter`.

After copying this directory into a project:

1. Create a MySQL or MariaDB database and user.
2. Open the project URL and complete WordPress's installation form. WordPress will create `wp-config.php`, or you can copy `wp-config-sample.php` and enter the database credentials manually.
3. In **Appearance > Themes**, activate **Starter**.
4. Do not commit `wp-config.php`, uploads, caches, database dumps, or secrets.

Keep WordPress core and plugins updated independently in every active project.
