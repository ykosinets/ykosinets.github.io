# Shopify starter

This is Shopify's official Skeleton Theme.

After copying this directory into a project:

1. Install the current Shopify CLI.
2. Sign in to a Shopify partner/development store with theme permissions.
3. Run `shopify theme dev --store your-store.myshopify.com` from the copied directory.
4. Run `shopify theme check` before sharing or deploying changes.
5. Use `shopify theme push --unpublished` for a client-review theme; publish only after approval.

Shopify store credentials and CLI state must not be committed.
