# Payload starter

This is Payload's official blank template configured for local SQLite.

After copying this directory into a project:

1. Run `npm install`.
2. Copy `.env.example` to `.env`.
3. Replace `PAYLOAD_SECRET` with a long random value.
4. Run `npm run dev` and open `http://localhost:3000/admin`.
5. Create the first administrator account.

The local `payload.db`, `.env`, uploaded media, dependencies, and Next.js build output must not be committed.
